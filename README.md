
# Telegram Mines Bot

Advanced Telegram Bot for Mines Game with:

- Auto Signals 💎  
- VIP System 👑  
- Verify Tool 🔐

## Commands

- `/start` - Welcome  
- `/signal` - Demo signal  
- `/verify server client nonce` - Calculate result  
- `/addvip user_id` - Add VIP (Admin only)  
- `/removevip user_id` - Remove VIP (Admin only)

## Setup

1. Add BOT_TOKEN as environment variable  
2. Set ADMIN_ID, FREE_CHANNEL, VIP_CHANNEL in bot.py  
3. Run `python bot.py`
